# Week 03 - Joins

## Overview

This lab focuses on retrieving data from multiple tables using SQL joins. The exercises cover combining related tables, matching records with different join techniques, and generating reports using the Oracle HR sample schema.

## Files

* `Lab03_joins.sql` — Lab solutions
* `Week03.pdf` — Lab instructions (with instructor permission)

## Concepts Covered

* Cross Join
* Natural Join
* USING Clause
* ON Clause
* Left Outer Join
* Right Outer Join
* Full Outer Join
* Self Join
