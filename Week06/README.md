# Week 06 - Data Manipulation Language (DML)

## Overview

This lab focuses on SQL Data Manipulation Language (DML) statements. The exercises demonstrate how to insert, update, delete, commit, and roll back data while working with database tables.

## Files

* `lab06_dml.sql` — SQL solutions for the lab exercises
* `Week06-DML.pdf` — Lab instructions (with instructor permission)

## Concepts Covered

* INSERT Statement
* UPDATE Statement
* DELETE Statement
* COMMIT
* ROLLBACK
* Creating Tables
* Data Manipulation Operations
