# Week 04 - Group Functions

## Overview

This lab focuses on SQL aggregate functions and grouping data. The exercises demonstrate how to summarize data using aggregate functions, organize records with the `GROUP BY` clause, and filter grouped results using the `HAVING` clause.

## Files

* `lab04_group_functions.sql` — SQL solutions for the lab exercises
* `lab04.pdf` — Lab instructions (with instructor permission)

## Concepts Covered

* Aggregate Functions (`COUNT`, `SUM`, `AVG`, `MIN`, `MAX`)
* GROUP BY Clause
* HAVING Clause
* Aggregate Calculations
* Grouping Records
* Filtering Groups
* Sorting Grouped Results
