# Week 09 - Constraints

## Overview

This lab focuses on SQL Constraints in Oracle Database. The exercises demonstrate how to create and manage table constraints to maintain data integrity and enforce relationships between tables.

## Files

* `lab09_constraints.sql` — SQL solutions for the lab exercises
* `Week09 - Constraints.pdf` — Lab instructions (with instructor permission)

## Concepts Covered

* PRIMARY KEY Constraint
* FOREIGN KEY Constraint
* CHECK Constraint
* ALTER TABLE
* ADD COLUMN
* USER_CONSTRAINTS Data Dictionary
* Data Integrity
* Referential Integrity
