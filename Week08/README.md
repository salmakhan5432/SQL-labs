# Week 09 - Data Definition Language (DDL)

## Overview

This lab focuses on SQL Data Definition Language (DDL). The exercises demonstrate how to create, modify, rename, and remove database objects while working with Oracle data dictionary views.

## Files

* `lab08_ddl.sql` — SQL solutions for the lab exercises
* `Week9-DDL.pdf` — Lab instructions (with instructor permission)

## Concepts Covered

* CREATE TABLE
* ALTER TABLE
* INSERT INTO ... SELECT
* USER_TABLES Data Dictionary
* CREATE TABLE AS SELECT (CTAS)
* DROP TABLE
* RENAME
* DROP COLUMN
* DESCRIBE Table Structure
