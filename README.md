# SQL Labs

This repository contains my Database Systems lab solutions from the 3rd semester of my BS Software Engineering program. The purpose of this repository is to document my learning journey, practice SQL concepts, and maintain an organized collection of weekly lab work.

The labs and solutions in this repository are implemented using Oracle Database as part of the Database Systems course.

## Repository Structure

```text
SQL_Labs/
├── Week01/
├── Week02/
├── Week03/
├── Week04/
└── ...
```

Each week contains:

* SQL solution file(s)
* Lab instructions (if permitted by the instructor)
* A README explaining the lab topics and objectives

## Weekly Labs

| Week    | Topic                          |

| Week 02 | Selection Operation & ORDER BY |
| Week 03 | Joins                          |

Throughout the semester, this repository will include:

* Projection & Selection Operations
