# Week 05 - Subqueries

## Overview

This lab focuses on SQL subqueries. The exercises demonstrate how to retrieve data using nested `SELECT` statements, compare values returned by subqueries, and work with both single-row and multiple-row subqueries.

## Files

* `lab05_subqueries.sql` — SQL solutions for the lab exercises
* `Week5-Subqueries.pdf` — Lab instructions (with instructor permission)

## Concepts Covered

* Single-Row Subqueries
* Multiple-Row Subqueries
* Nested `SELECT` Statements
* Comparison Operators (`=`, `>`, `<`, `IN`)
* Filtering Data with Subqueries
* Department-Based Queries
* Manager and Employee Relationships
