# Week 08 - Views

## Overview

This lab focuses on SQL Views. The exercises demonstrate how to create, retrieve, modify, and delete views while controlling data access and enforcing data integrity using view constraints.

## Files

* `lab07_views.sql` — SQL solutions for the lab exercises
* `Week7.pdf` — Lab instructions (with instructor permission)

## Concepts Covered

* CREATE VIEW
* Simple Views
* Complex Views
* Retrieving Data from Views
* Data Dictionary (`USER_VIEWS`)
* WITH CHECK OPTION
* Updating Data Through Views
* DROP VIEW
